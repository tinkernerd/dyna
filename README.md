# ğŸ³ Docker Support (Added by @tinkernerd)

Run the entire portfolio with one command.

### **docker-compose.yml**

```yaml
services:
  dyna-photosite:
    build:
      context: .
      dockerfile: Dockerfile
    container_name: dyna-photosite
    restart: unless-stopped

    ports:
      - "3000:3000"

    volumes:
      - ./data:/data
```

### **Run it**

```bash
docker compose up --build -d
```

Open:

```
http://localhost:3000
```

### **All user content lives in `/data`**

* `data/photos` â€” your albums
* `data/thumbs` â€” auto-generated
* `data/about.md` â€” About page
