# Features

- **Autogenerated albums** | Every folder in `/photos` becomes an album page
- **Auto updating** | Add or remove photos from folders and the site will update after a page refresh
- **EXIF metadata display** | Shows off your camera specs
- **Can embed "location data"** | Include "Location: Location name" in the descriptor file to have a location tag
- **Fullscreen image viewer** | Click or tap to view full screen photo
- **Chronological sorting** | Photos are sorted by file creation time
- **No database and no CMS**
- **Runs anywhere** | Static Node.js server, host locally or on VPS
- **Lazy loading** | Photos load in order instead of all at once
- **Preview photos are compressed** | New uploaded photos will generate a compressed thumbnail to save bandwidth. Full rez photos only loaded when viewing in full screen
- **Customizable about page** | Edit the about.md file, supports MD

# Install

1. Clone the repo
2. `npm install`
3. Replace stock photos, following this structure:
```
photos/
  ├── album/
  │     ├── img1.jpg
  │     ├── img2.jpg
  │     └── ...
```

5. run `node server.js` and open `http://localhost:3000`
